# PVH Schule

Automated redirect and landing page generator for Netlify.

## Overview
This project uses GitHub Actions to automatically build and deploy a static site and redirect rules.

## Workflow
1.  Edit `body.txt` to manage redirects.
2.  Edit `header.md` to update the landing page appearance.
3.  Push changes to GitHub to trigger the deployment.

## Local Build
Run `make` to generate the output folder `weiterleitung` locally.
